# 🎓 Student Data Organizer

A simple command-line application written in Python to manage student records. This project demonstrates the use of Python data structures such as **lists, dictionaries, tuples, and sets**, along with loops, conditional statements, and user input.

## 📌 Features

- ➕ Add new students
- 📋 Display all student records
- ✏️ Update student information
- ❌ Delete student records
- 📚 Display all unique subjects offered
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3

## 📂 Data Structures Used

| Data Structure | Purpose |
|---------------|---------|
| List | Stores all student records |
| Dictionary | Stores each student's information |
| Tuple | Stores Student ID and Date of Birth |
| Set | Stores unique subjects |

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/student-data-organizer.git
```

2. Navigate to the project folder:

```bash
cd student-data-organizer
```

3. Run the program:

```bash
python student_data_organizer.py
```

> Replace `student_data_organizer.py` with your actual Python file name if it is different.
